Vagrant / Node / Express / Mongo

Vagrant setup to get a Node / Mongo stack up and running.

The application itself is based on the tutorial of Christopher Buecheler (http://cwbuecheler.com/web/tutorials/2013/node-express-mongo/)

This code generate 2 VMs ('app' & 'db').

Requirements

  • Vagrant V1.2.2 min (download and install from http://www.vagrantup.com/downloads.html)
  • Virtualbox V4.2.12 min (download and install from https://www.virtualbox.org/wiki/Downloads)

Add a standard box

vagrant box add precise32 http://files.vagrantup.com/precise32.box

Get the repository

git clone git@github.com:jgrossiord/vagrant-node-mongo.git

Start the VM

cd vagrant-node-mongo
vagrant up

This will startup the 2 VM's. For each VM, if you have more than one network card on your PC, it will ask which card you want to bridge to the VM. Each VM will have a private IP (10.11.12.2 & 10.11.12.3) which enable the app server to contact Mongo directly and a bridge address, taken from you DHCP server, enabling you and your co-workers to connect to the VM's. At the end of the startup process, the URL of the application will be shown.

Other commands (see Vagrant doc)

To suspend the 2 VM

vagrant suspend

To destroy the 2 VM

vagrant destroy

Connect in SSH to the VM's

vagrant ssh app
vagrant ssh db
